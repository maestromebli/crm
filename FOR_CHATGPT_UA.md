# Бекап коду для ChatGPT (аналіз CRM)

**Результат аналізу та чеклист покращень:** [docs/GPT_ANALYSIS_AND_ACTIONS.md](./docs/GPT_ANALYSIS_AND_ACTIONS.md)  
**Дубль інструкції (docs):** [docs/BACKUP_FOR_CHATGPT.md](./docs/BACKUP_FOR_CHATGPT.md)

## Як згенерувати свіжий архів

У корені проєкту:

```bash
pnpm backup:gpt
```

Після запуску з’являться:

| Файл | Призначення |
|------|-------------|
| `backups/gpt-analysis-<дата-час>/crm-for-chatgpt-analysis.tar.gz` | Архів коду та конфігів **без** `node_modules`, `.next`, `.git`, `.env*` |
| `backups/crm-for-chatgpt-analysis-<дата-час>.zip` | Той самий зміст у ZIP (зручно для завантаження в ChatGPT на Windows) |
| `backups/gpt-analysis-<дата-час>/README_FOR_CHATGPT.md` | Коротка інструкція до архіву |

**Шлях до останнього архіву** після кожного запуску також записується в **`backups/LATEST_GPT_BUNDLE.txt`**.

## Що завантажити в ChatGPT

1. Файл **`crm-for-chatgpt-analysis.tar.gz`** (або розпакуйте й прикріпіть окремі папки `src/`, `prisma/`, якщо ліміт розміру не дозволяє весь архів).
2. За потреби — цей файл **`FOR_CHATGPT_UA.md`** або текст промпту нижче.

Секрети (паролі БД, ключі API) **не** входять у архів — це навмисно.

## Промпт для аналізу та покращення (скопіюйте в чат)

```
Ти — senior-архітектор продукту та коду. У вкладенні — архів Next.js CRM (ліди, угоди, контакти, Prisma/PostgreSQL, API routes).

Завдання:
1) Коротко опиши архітектуру (App Router, шари lib/features/components, авторизація).
2) Виділи сильні сторони та технічний борг / ризики.
3) Запропонуй 5–10 конкретних покращень (пріоритет: безпека, підтримуваність, UX, продуктивність) з посиланнями на файли/модулі з архіву.
4) Якщо є питання по домену — сформулюй припущення явно.

Відповідай українською, структуровано, без загальних фраз без прив’язки до коду.
```

## Повний бекап разом із БД

Якщо потрібен ще дамп PostgreSQL (обережно з персональними даними):

```bash
pnpm backup:full
```

Не публікуйте повний дамп без видалення чутливих даних.
